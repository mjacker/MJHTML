BLOCK and Inline Elements

  • Un bloque siempre empieza en una nueva linea, y toma todo el ancho disponible.

    <div style="border: 1px solid black">

  • Un elemento en linea no inicia en una nueva linea, y toma solo el ancho que necesita.

    <span style="border: 1px solid black">

Class Attribute, Id Attribute

  • Class Attribute: 
    
      <style>
      .city {
        background-color: tomato;
        color: white;
        border: 2px solid black;
        margin: 20px;
        padding: 20px;
      }
      </style>
    <div class="city">

    <h2>London<h2>

    <p>London is the capital of England.</p>

    </style>

    Los atributos de clases de "city" dentro de <div>, reemplaza las propiedades de <h2> y de <p>. Tambien sera afectado los demas <div> que tengan el nombre de clase "city".
  • Id Attribute: 
    
          <style>
          #miCabeza {
            background-color: lightblue;
            color: black;
            padding: 40px;
            text-align: center;
          }
          </style>

    < h1 id="miCabeza"</h1>

    < p id="miCabeza"</h1>

    • Solo afecta a las etiquetas que tengas exactamente el nombre de id ya pre-establecido. No puedo afectar a un bloque de etiquetas.
