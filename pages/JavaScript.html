JavaScript - noscript

    Si el navegador no soporta JavaScript, puedes mostrar un mensaje con <noscript>

            <script>
          document.getElementById("demo").innerHTML = "Tu navegador soporta JavaScript!";
        </script>

        <noscript>Lo siento, tu navegador no soporta JavaScript.!</noscript>

JavaScript Output

  • innerHTML

    Para acceder a un elemento se usa método document.getElementById(id).

    El id define el atributo del elemento. El innerHTML define el contenido.

              <p id="innerHTML">

          <script>
            document.getElementById("innerHTML").innerHTML = 5 + 6;
          </script>

  • document.write()

    Solo para realizar pruebas,

    Si se utiliza despues de cargar el HTML, borrará todo el contenido!.

              <script>
            document.write(5+6);
          </script>

  • window.alert()

    Para mostrar una alerta desde un cuadro de diálogo.

            <button type="button" onclick="alerta()">Click aqui para mostrar una alerta!.</button>
    
        <script>
          function alerta(){
            window.alert("ESTO ES UNA ALERTA!!!.");
          }
        </script>
    El keyword window puede ser omitido.
    alert("ESTO ES UNA ALERTA!!!."); 
              <script>
          function alerta(){
            alert("ESTO ES UNA ALERTA!!!.");
          }
      </script>

  • console.log

    Es con el propósito de depurar, y se utiliza en la consola del navegador para mostrar datos.

              <script>
            console.log("Texto en consola.")
          </script>

  • window.print()

    No se puede acceder a disposivos de salidas desde JavaScript, a exepción de un método para imprimir el contenido de la ventana actual.
