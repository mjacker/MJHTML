HTML TABLAS

TH para titulo de tabla, style="border: 2px solid black;"
TR es para crear una nueva linea. TD es para crear una celda.

Agregar un borde

    <table style="border:1px solid black">

Bordes colapsados

    titulo, utilizando style="border-collapse: collapse;"
    TEXTO 1 TEXTO 2

Bordes NO colapsados

    titulo, utilizando style="border-collapse: collapse;"
    TEXTO 1 TEXTO 2

Agregar rellenos a las celdas (Padding)

    style="padding:30px" style="padding:30px"
    style="padding:60px" style="padding:60px"

Alineaacion de Titulos (left-right)

    style="text-align: left"
    Esto es solo un texto dentro de esta celda.

    style="text-align: right"
    Esto es solo un texto dentro de esta celda.

Agregar espacios entre celdas

    <table style="border-spacing: 30px">
    TITULO 1
    TEXTO 1

Celdas que ocupan varias filas o columnas.

    Nombre Telefonos. <th colspan="2">
    Jaime Colmán 0992250360 098444543

    Nombre Telefonos
    Jaime Colmán,
    <th rowspan="2">    		 099252360
    0984424543
