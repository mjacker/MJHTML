HTML Links

  • a href="https://www.google.com"
  • a href="https://www.google.com" target="_self" Abre en la misma pestaña.
  • a href="https://www.google.com" target="_blank" Abre en una nueva ventana o pestaña.
  • a href="https://www.google.com" target="_parent" Abre en un frame.
  • a href="https://www.google.com" target="_top" (No me funciona)
  • Esta imagen es un enlace a Google.
  • href="mailto:someone@example.com" Enviar un correo.
  • FALTA CORREGIR.
  • Se puede agregar un estilo con color de fondo a los enlaces usando CSS. 
    <style>
    a:link, a:visited {
    background-color: #f44336;
    color: white;
    padding: 15px 25px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    }

    a:hover, a:active {
    background-color: red;
    }
</style>

URLs absolutos y relativos

  • a href="https://www.google.com" Enlace absoluto, la dirección completa a otra pagina.
  • a href="/carpeta/NombreArchivo.html" Enlace a otra pagina alojado en la misma web.

HTML Bookmarks

  • h2 id="C4"

  • a href="#C4" Salta donde se encuentra el ID="C4".
